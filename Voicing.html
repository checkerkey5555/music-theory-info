             SUBSTITUTIONS FOR MINOR CHORDS
             ok so you have a 2-5-1 and for the Minor 2 you want alternate chords to use here is some stuff you can use
             instead                                                     
 
               Minor7th Block Chord: R-M3-5-6                                                 
               Minor9th Chord: R-3-5-7-9 (sometimes the root is omitted in a lefthand 2-5-1)  
               Minor7th Flat5th: R-M3-M5-M7                                                   

             SUBSTITUTIONS FOR DOMINANT CHORDS
             ok so you have a 2-5-1 and for the Dominant 5 you want alternate chords to use some stuff you can use instead                                                     

               Dominant7th Flat5th: R-3-M5-M7                                                 
               Dominant7th#5t: R-3-M6-M7                                                      
               Dominant7thSus4th: R-4-5-7                                                     
               Dominant9th: R-3-5-M7-9                                                        
               Dominant11th: R-3-5-M7-9-11                                                    
               Dominant13th: R-3-5-M7-9-11-13                                                 

             SUBSTITUTIONS FOR MAJOR CHORDS
             ok so you have a 2-5-1 and for the Major 1 you want alternate chords to use some stuff you can use instead                                                     

               Major7th Block Chord: R-3-5-6                                                  
               Major9th Chord: R-3-5-7-9 (sometimes the root is omitted in a lefthand 2-5-1)  
               Major11th Chord: R-3-5-7-9-11                                                  
               Major13th Chord: R-3-5-7-9-11-13                                               
               Major6/9 Chord: R-3-5-6-9                                                      
               Major Lydian Chord: R-3-5-M7-(9)-#11-(13)
